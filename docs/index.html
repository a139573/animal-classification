Readme Practice 1

In this README, we will explain where the data was obtained from, as well as list the libraries necessary to execute the files and how to run this project.

Data Source

The dataset was obtained from Kaggle. The dataset is titled Animal Image Dataset (90 Different Animals). The dataset is available at the following URL: https://www.kaggle.com/datasets/iamsouravbanerjee/animal-image-dataset-90-different-animals. This dataset contains 5400 animal images across 90 classes.

Libraries to import

  • Torch
  • Torchvision
  • PIL
  • pathlib

What the project does

The goal of this project is to develop a model capable of identifying which of the 90 animals is depicted in a given image. The model will be trained on a portion of the dataset to learn features that distinguish between different animals. Once trained, the model should be able to accurately classify previously unseen images.

How to run the project

Download the project folder and open the Jupyter Notebook named Project 1 Animals Image Dataset. Run all cells to execute the project.

How to train the model

Run the following command from the project root folder: python -m my_projects.modeling.train. This trains the model on a subsample containing 6% of the data.

How to generate predictions

Run the following command from the project root folder: python -m my_projects.modeling.inference.

Environment used

The project environment is Anaconda. The environment py311ml was used.
